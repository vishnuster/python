to configure git for the first time:

git config --global user.name "your username"
git config --global user.email "your email ID"


To add a remote repository: git remote add origin https://github.com/vishnuster/test.git


to initiate a repo: change directory to the folder name cd <folder name> and run : git init

to check if the remote repo has been added:  git remote -v

to make git recognise a file: git add <filename>

to commit changes to repo: git commit -m "commit comment"

to push commits to remote repo: git push <branch name>

to pull commits from remote repo: git pull

to remove a file from repo: git rm <file name>


Amending older or multiple commit messages

If you have already pushed the commit to GitHub, you will have to force push a commit with an amended message.

#Amending the message of the most recently pushed commit : git push --force example-branch
